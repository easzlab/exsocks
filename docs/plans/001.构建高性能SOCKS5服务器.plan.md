# 高性能 SOCKS5 代理服务器实现计划

## 项目结构

```
src/
├── main.rs              # 程序入口、命令行解析
├── config.rs            # 配置管理
├── server.rs            # TCP 服务器核心逻辑
├── socks5/
│   ├── mod.rs           # 模块导出
│   ├── protocol.rs      # SOCKS5 协议常量和类型定义
│   ├── handshake.rs     # 握手和认证协商
│   ├── request.rs       # 请求解析 (CONNECT 命令)
│   └── reply.rs         # 响应构建
├── relay.rs             # 零拷贝数据转发
```

## Task 4: 握手处理

**文件**: `src/socks5/handshake.rs`

实现握手流程（RFC1928 第3节）：
1. 读取客户端版本标识和方法选择消息
2. 验证 VER = 0x05
3. 检查是否支持无认证方法 (0x00)
4. 发送方法选择响应

```rust
pub async fn perform_handshake(stream: &mut TcpStream) -> Result<(), SocksError>
```

## Task 5: 请求解析

**文件**: `src/socks5/request.rs`

解析 CONNECT 请求（RFC1928 第4节）：
1. 读取 VER, CMD, RSV, ATYP
2. 验证 CMD = 0x01 (CONNECT)
3. 根据 ATYP 解析目标地址：
   - 0x01: 读取 4 字节 IPv4
   - 0x03: 读取 1 字节长度 + 域名
   - 0x04: 读取 16 字节 IPv6
4. 读取 2 字节端口号

```rust
pub struct ConnectRequest {
    pub address: Address,
    pub port: u16,
}

pub async fn parse_request(stream: &mut TcpStream) -> Result<ConnectRequest, SocksError>
```

```rust
pub async fn relay(
    client: TcpStream,
```

- 使用 `copy_bidirectional` 避免手动缓冲区管理

## Task 12: 服务器主循环

**文件**: `src/server.rs`

```rust
pub async fn run(config: Config) -> Result<(), SocksError> {
    let listener = TcpListener::bind(config.bind_addr).await?;
    let limiter = Arc::new(ConnectionLimiter::new(config.max_connections));
    
    loop {
        let (socket, peer_addr) = listener.accept().await?;
        let limiter = limiter.clone();
        
        tokio::spawn(async move {
            if let Err(e) = handle_connection(socket, peer_addr, limiter).await {
                tracing::error!(%peer_addr, error = %e, "Connection error");
            }
        });
    }
}
```

## Task 13: 连接处理流程

**文件**: `src/server.rs`

```rust
async fn handle_connection(
    mut socket: TcpStream,
    peer_addr: SocketAddr,
    limiter: Arc<ConnectionLimiter>,
) -> Result<(), SocksError> {
    // 1. 获取连接许可（限流）
    let _permit = limiter.acquire().await?;
    
    // 2. SOCKS5 握手
    handshake::perform_handshake(&mut socket).await?;
    
    // 3. 解析 CONNECT 请求
    let request = request::parse_request(&mut socket).await?;
    
    // 4. 连接目标服务器
    let target = TcpStream::connect((request.address, request.port)).await?;
    let bind_addr = target.local_addr()?;
    
    // 5. 发送成功响应
    reply::send_reply(&mut socket, REP_SUCCEEDED, bind_addr).await?;
    
    // 6. 开始双向转发
    relay::relay(socket, target).await?;
    
    Ok(())
}
```
